ai_playground_model_usage

This table records the usage count of different AI models in the AI Playground. Usage is tracked per user and per model; that is, each user has a separate count for each model. The usage count is used to enforce quotas and is reset daily by a scheduled job.

Primary key constraint

  • pk1

Foreign key constraints

  1. ai_pg_usage_count_fk1 (On Delete Cascade)

Indexes

AI model for which usage count and quota are tracked, e.g. AMAZON_NOVA_MICRO, AMAZON_NOVA_LITE. This is represented as an int for easier querying and indexing, and the mapping of int to model name is maintained in the application code. The default 0 denotes None for backward compatibility.
usage_count int 1 false false

The number of times the user has used the model in the AI Playground. This count is used to enforce usage quotas.
