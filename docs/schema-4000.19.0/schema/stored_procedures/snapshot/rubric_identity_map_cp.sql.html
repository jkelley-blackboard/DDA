rubric_identity_map_cp

This stored procedure seeds identity rows (old_pk1 = new_pk1) into tmp_key_maps for rubric and its rubric_row, rubric_column, and rubric_cell descendants. Accepts either a single rubric PK or a course PK to process all rubrics in the course. Used during same-course copy to avoid duplicating rubrics.
