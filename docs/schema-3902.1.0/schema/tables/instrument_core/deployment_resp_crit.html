deployment_resp_crit

This table records the relationship between each response record in deployment_response and the records in deployment_criteria that resulted in the creation of this response.

Primary key constraint

deployment_resp_crit_pk

This is the primary key constraint on this table.

  • pk1

Foreign key constraints

  1. deployment_resp_crit_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [INSTRUMENT_CORE].deployment_response table, referential action cascade on delete.

    • deployment_response_pk1
  2. deployment_resp_crit_fk2

    This is a foreign key referencing the primary key of the [INSTRUMENT_CORE].deployment_criteria table. Deletes will not cascade; records must first be directly deleted from this table before deleting referenced records from deployment_criteria.

    • deployment_criteria_pk1

Indexes

  2. deployment_resp_crit_if2
    • deployment_criteria_pk1
  3. deployment_resp_crit_ak1
    • deployment_response_pk1
    • deployment_criteria_pk1
