materialized_list

A materialized list is a list of users, the product of the execution of a list definition. A deployment is tied to at least one materialized list. These lists are timestamped on creation, and can optional be assigned a name and description as well.

Primary key constraint

materialized_list_pk1

This is the primary key on column pk1 of this table.

Foreign key constraints

  1. materialized_list_fk1 (On Delete SetNull)

    This is a foreign key referencing the primary key of the list_definition table, referential action cascade on delete (setnull).
