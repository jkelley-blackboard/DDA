deployment_response

This table keeps track of status of responses for each recipient in a deployment. Response records are created for each recipient at the deployment strat time, and updated when a response is received.

Primary key constraint

deployment_response_pk1

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. deployment_response_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the deployment table, referential action cascade on delete (cascade).

    • deployment_pk1
  2. deployment_response_fk2

    This is a foreign key referencing the primary key of the materialized_list table.

    • materialized_list_pk1
  3. deployment_response_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • user_pk1
  4. deployment_response_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the [AS_CORE].course_main table, referential action on delete (setnull).

    • crsmain_pk1
