domain_pk1 id false false

This is the Foreign Key referencing domain table that is enforced in the code not at the database level. It is the domain that the institution role is linked to. A domain can have many institution roles.
institution_role_pk1 id false false

This is the Foreign Key referencing institution_role table that is enforced in the code not at the database level. It is the institution role that is linked to the domain. A institution role can be linked to multiple domains.
