ods_aa_content_activity

This table is a result of the materialization of ods_aa_content_activity_vw. The data is loaded through ETL process. It lists content access records aggregated at the content level within each login session.

Primary key constraint

ods_aa_content_activity_pk

  • login_pk1
  • course_pk1
  • content_pk1

Indexes

  1. ods_aa_content_activity_ie1
    • user_pk1
  2. ods_aa_content_activity_ie2
    • course_pk1
  3. ods_aa_content_activity_ie3
    • content_pk1
