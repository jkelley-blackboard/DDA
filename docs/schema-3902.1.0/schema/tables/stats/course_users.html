course_users

This table contains information on user enrollment in courses. It also include user role assignment within courses. When directly accessing the database to UPDATE a course_users record, you need to call sync_master_course_user stored procedure with the pk1 of the modified course_users record so that, if needed, proper replication occurs to the master record (Cross Listing/Merged Enrollment).
