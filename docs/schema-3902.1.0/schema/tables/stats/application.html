application

This table contains general information on a list of tools (addressbook, gradebook etc.) that are available in the system.

Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
application varchar(64) false false

This is the handle of the tool, used to reference the tool in other database tables/code.
large_icon varchar(255) false true

This is the image file name for the icon that displays on the UI for the tool. 50 x 50 pixels.
plugins_pk1 id false true

This is a Foreign Key referencing the primary key of the Building Block that created the tool (if any).
proxy_tools_pk1 id false true

This is a Foreign Key referencing the primary key of the Proxy Tool that created the tool (if any).
blti_placement_pk1 id false true

This is a Foreign Key referencing the primary key of the Basic LTI placement that created the tool (if any).
enabled_mask int false true

This the bit mask that determines if the tool is enabled or not. SYSTEM = 0x00000001, COURSE = 0x00000002, ORG = 0x00000004, COURSE_GROUP = 0x00000008, ORG_GROUP = 0x00000010, NAVITEM_COURSE = 0x00000001, NAVITEM_ORG = 0x00000002, NONE = 0x00000000, ALL = 0xffffffff.
prev_enabled_mask int false true

The previous configured value of enabled_mask. This is used when enabling/disabling Building Blocks so that when a Building Block is disabled, then enabled, the previous customizations are retained.
