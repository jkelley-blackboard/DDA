delegated_grader

Lists the specific delegated graders for a given gradebook column.

Primary key constraint

delegated_grader_pk

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. delegated_grader_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the gradebook_main table. (When the column is deleted, delete all delegated graders)

    • gradebook_main_pk1
  2. delegated_grader_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the users table. (When the grader is deleted, delete this row - no need to retain them as a delegated grader)

    • users_pk1

Indexes

  1. delegated_grader_if2
    • users_pk1
  2. delegated_grader_ak1
    • gradebook_main_pk1
    • users_pk1
