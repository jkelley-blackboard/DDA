delta_due_date int false true

Indicates whether the user or group has a due date exception. A value equal to zero means the item will never be marked as late, and a value not equal to zero means the new due date is the gradable item's due date plus this value, which specifies a duration in minutes (can be positive or negative). If this value and fixed_due_date are both null, then there is no due date exception. Both cannot be simultaneously non-null.
fixed_due_date datetime false true

Indicates whether the user or group has a due date exception. A non-null value represents the new due date for this user/group's gradable item. If this value and delta_due_date are both null, then there is no due date exception. Both cannot be simultaneously non-null.
due_date_exception_type char(1) 'N' due_date_exception_type_enum
  • N

    Normal: No due date exception (delta_due_date and fixed_due_date are both null).

  • F

    Fixed: Due date exception is static or fixed (fixed_due_date is non-null).

  • R

    Relative: Due date exception is relative to the gradable item's due date (delta_due_date is non-null and non-zero).

  • U

    Unlimited: No due date; the item should never be marked as late (delta_due_date is zero).
