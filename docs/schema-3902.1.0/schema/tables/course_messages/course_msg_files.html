course_msg_files

This table associates messages and the standalone file attachment each may contain. Each message can have at most 1 file attachment.

Primary key constraint

course_msg_files_pk

This is the primary key of this table.

  • pk1

Foreign key constraints

  1. course_msg_files_fk1

    This is a foreign key referencing pk1 of the course_msg table. When a message is deleted, any attachment will be removed in the code.

    • course_msg_pk1
  2. course_msg_files_fk2

    This is a foreign key referencing pk1 of the files table.

    • files_pk1

Indexes

  1. course_msg_files_ak1

    A message can have at most 1 standalone attachment.

    • course_msg_pk1
  2. course_msg_files_if1
    • files_pk1
