course_msg_migration

This table is used during flat file to relational Course Message migration as a lookup a flat file id (e.g UUID ) to a relational pk of a migrated Course Message, Personal Folder or Conversation. The lookup data is used to determine whether a Message/Personal Folder/Conversation has already been migrated

Primary key constraint

course_msg_migration_pk

This is the primary key of this table.

  • pk1

Indexes

  1. course_msg_migration_ak1

    Ensure a flat file UUID for a Course Message or Conversation is migrated once. Note, when a Course is copied or restored, the the file UUIDs don't change so we limit the unique check here per course.
