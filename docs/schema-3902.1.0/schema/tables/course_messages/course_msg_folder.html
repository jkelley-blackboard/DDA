course_msg_folder

This table contains information on Learn Course and Ultra message folders. Ultra messages, like Course Messages, are at the course level, but supports future extension for messages sent outside the course context. Essentially, Ultra Messages is Course Messages with a different UI.

Primary key constraint

course_msg_folder_pk

This is the primary key of this table.

  • pk1

Foreign key constraints

  1. course_msg_folder_fk1 (On Delete Cascade)

    This is a foreign key referencing pk1 of the course_main table.

    • crsmain_pk1
  2. course_msg_folder_fk2 (On Delete Cascade)

    This is a foreign key referencing pk1 of the users table.

    • users_pk1

Indexes

  1. course_msg_folder_if1
    • crsmain_pk1
    • users_pk1
  2. course_msg_folder_if2
    • users_pk1

default_ind char(1) 'N' crs_msg_fldr_def_ind_ck
  • Y

    This is a default folder, which is auto-created by LEARN and has a localized label, e.g. 'Inbox' and 'Sent'.

  • N

    This is a custom folder, which are created by the course user.

 false false

An indicator for whether the folder is a default or a custom folder.
label nvarchar(255) false false

For default folders, the label is a resource key. And for custom folders, the labels are what the user entered.
crsmain_pk1 id false true

This is a foreign key to the course_main table, i.e. course_main.pk1. For default folders, this is NULL. For custom folders, it is the course in which the custom folder is created. This is also NULL for messages sent outside a course (future requirement).
users_pk1 id false true

This is a foreign key to the users table, i.e. users.pk1. For default folders, this is NULL. For custom folders, it is the pk1 of the user who created the folder.
