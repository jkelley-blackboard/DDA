course_note

Table holding notes against course student memberships

Primary key constraint

course_note_pk1

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. course_note_fk1 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table for the author relationship.

    • author_users_pk1
  2. course_note_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the course_users table for the student relationship.

    • subject_course_users_pk1
  3. course_note_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table for the last modified user relationship.

    • modified_by_users_pk1

Indexes

    • subject_course_users_pk1
  2. course_note_if2
    • author_users_pk1
