clp_sog_activity

This tables records activities relate to assessment project objective.

Primary key constraint

clp_sog_activity_pk1

This is the name of the primary key constraint on the column pk1.

  • pk1

Foreign key constraints

  1. clp_sog_activity_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the clp_sog table, referential action cascade on delete (cascade).

    • clp_sog_pk1
  2. clp_sog_activity_fk2

    This is a foreign key referencing the primary key of the form table.

    • form_pk1
  3. clp_sog_activity_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • modifying_user_pk1
