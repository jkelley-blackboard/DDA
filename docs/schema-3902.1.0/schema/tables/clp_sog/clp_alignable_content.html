clp_alignable_content

This table identifies a piece of course content that is aligned to a SOG. Keep FK constraints in synch with StandardAlignmentDataHandler

Primary key constraint

clp_align_content_pk1

This is the name of the primary key constraint on the column pk1.

  • pk1

Foreign key constraints

  1. clp_align_content_fk1 (On Delete SetNull)

    This is a foreign key referencing the primary key of the blogs table, referential action cascade on delete (setnull).

    • blogs_pk1
  2. clp_align_content_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the forum_main table, referential action cascade on delete (setnull).

    • forum_main_pk1
  3. clp_align_content_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the qti_asi_data table, referential action cascade on delete (setnull).

    • qti_asi_data_pk1
  4. clp_align_content_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the msg_main table, referential action cascade on delete (setnull).

    • msg_main_pk1
  5. clp_align_content_fk5 (On Delete SetNull)

    This is a foreign key referencing the primary key of the gradebook_main table, referential action cascade on delete (setnull).

    • gradebook_main_pk1
  6. clp_align_content_fk6 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_contents table, referential action cascade on delete (setnull).

    • course_contents_pk1
  7. clp_align_crsmain_pk1_fk7 (On Delete Cascade)

    This is a foreign key referencing the primary key of the course_main table, referential action cascade on delete (cascade).

    • crsmain_pk1
  8. clp_align_content_fk8 (On Delete SetNull)

    This is a foreign key referencing the primary key of the rubric_row table, referential action cascade on delete (setnull).

    • rubric_row_pk1
  9. clp_align_content_fk9 (On Delete SetNull)

    This is a foreign key referencing the primary key of the rubric table, referential action cascade on delete (setnull).

    • rubric_pk1

Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
blogs_pk1 id false true

This is a foreign key referencing the primary key of the blogs table. It identifies a blog that directly linked to a course.
forum_main_pk1 id false true

This is a foreign key referencing the primary key of the forum_main table. It identifies a forum that is directly linked to a course.
qti_asi_data_pk1 id false true

This is a foreign key referencing the primary key of the qti_asi_data table. It identifies a test or a test question. When this column is present, the course_contents_pk1 column is null.
gradebook_main_pk1 id false true

This is a foreign key referencing the primary key of the gradebook_main table. It identifies a grade book column.
msg_main_pk1 id false true

This is a foreign key referencing the primary key of the msg_main table. It identifies a forum thread.
course_contents_pk1 id false true

This is a foreign key referencing the primary key of the course_content table. It identifies a course content item.
rubric_row_pk1 id false true

This is a foreign key referencing the primary key of the rubric_row table. It identifies a single row in a rubric.
rubric_pk1 id false true

This is a foreign key referencing the primary key of the rubric table. It identifies a rubric that is associated to the course.
cms_pk1 id false true

This is a key referencing the Content System (cms) permanent id. It identifies a single cms content item
title nvarchar(100) false true

This is the title of each aligned item.
description nvarchar(100) false true

This is the description of the alignment.
type nvarchar(100) false true

This is the type of alignment.
crsmain_pk1 id false true

This refers the primary key of the course_main table.
ref_id varchar(255) false true

This is the ref_id.
dtcreated datetime NOW false false

This is the creation time. Used for cleaning up orphans -- make sure at least x minutes old so don't delete temp content.
dtmodified datetime NOW false false

Indicates when the content alignments were last modified. Updated automatically when changes are made to clp_content_alignment child entries.
extended_data image false true

Used to store arbitrary properties for this item
lor_clp_align_cnt_pk1 id false true

The alignable entry hosting the root Learning Object Repository item
lor_sync_date datetime false true

Time associated Learning Object Repository synchronization was last completed.
