The description is used only for alignment reports and it will be populated only if the row has alignments.

This is a Foreign Key referencing the pirmary key of course_main table.

This column is not used. It was originally intended to be a globally unique identifier following the IMS Persistent Location-Independent Resource Identifier recommendation but is NOT USED
position int false true

This is the position of where the item will be displayed on the UI.
data image false true

This is the QTI data for this question, section, or canvas.
bbmd_asi_type int false true

This is the qti asi type for this record. 1=Assessment, 2=Section, 3=Item
bbmd_assessmenttype int false true

This is the assessment type for this record when bbmd_asi_type=1. 1=Test, 2=Quiz (Not used), 3=Survey, 4=Pool, 5=Ultra Assignment (Not used)
bbmd_assessment_subtype varchar(255) false true

The assessment subtype is assigned to this record when bbmd_asi_type=1. Subtypes are utilized to further differentiate assessments optionally and provide the ability to define various types of Tests, Surveys, and Pools. For instance, a test may have a subtype of 'Assignment' in Ultra.
bbmd_questiontype int false true

This is the type of this record when bbmd_asi_type=3. 1=Multiple Choice, 2=True/False, 3=Multiple Anser, 4=Ordering, 5=Matching, 6=Fill In the Blank, 7=Essay Question, 8=Numeric, 9=Calculated, 10=FileUpload, 11=HotSpot, 12=FillintheBlankPlus, 13=Jumbled sentence, 14=Quiz Bowl, 15=LikeRT Scale, 16=Short Answer, 17=EitherOr
qmd_absolutescore_max float false true

This is the score that can be earned on this question item.
qmd_weighting float false true

This is the number of points given per question within this set/block when this record represents a random block or question set (bbmd_sectiontype=3 or 4)
bbmd_date_added datetime false true

This is the date time when the question item was added to the assessment.
bbmd_date_modified datetime false true

This is the date time when the question item was last modified.
bbmd_sectiontype char(1) false true

This is the type of this section then bbmd_asi_type=2. 1=Main Section, 2=Sub Section, 3=Random Block, 4=Question Block
bbmd_is_from_cartridge char(1) 'N' false true

This is the boolean flag (Y/N) that indicates if this question item comes from a cartridge.
bbmd_is_disabled char(1) 'N' false false

Flag for a disabled question - N means question is not disabled, Y means it is disabled
bbmd_keywords nvarchar(255) false true

This field is deprecated and should not be used.
bbmd_numbertype char(1) false true

This is the type of numbering to display (only applicable when the question has numbered items). 1=None, 2=Arabic, 3=Roman, 4=Upper Case Letters, 5=Lower case letters
bbmd_partialcredit char(1) false true

This is the boolean flag (Y/N) that indicates if this question may result in partial credit.
bbmd_orientationtype char(1) false true

This indicates the orientation of multiple options if applicable to the current question type. 1=Vertical, 2=Horizontal
bbmd_is_extracredit char(1) false true

This is the boolean flag (Y/N) that indicates if this question is for extra credit (Score is not included in the gradebook column's total score for the test)
bbmd_extra_credit_pts float false true

This is the number of extra credit points can be earned if bbmd_is_extracredit is true.
bbmd_negative_points_ind char(1) qad_negative_points_ck
  • Q
  • Y
  • N
false true

This is a flag to decide how negative points are handled: Y = yes and overall mark can go below 0. N = No. Q = only within the question so the net grade for the question cannot go below 0
bbmd_canvas_fullcrdt_ind char(1) qad_canvas_fullcrdt_ck
  • Y
  • N
false true

This flag indicates if this question (on this canvas only - overrides any manual or automatic grade if this is Y) should be graded at full marks all the time. Y==give full credit,N=grade normally
bbmd_all_fullcredit_ind char(1) qad_all_fullcredit_ck
  • Y
  • N
false true

This flag indicates if this question (everywhere it is used - overrides bbmd_canvas_fullcrdt_ind if this is Y) should be graded at full marks all the time. Y==give full credit,N=grade normally. Note that this feature is currently not implemented at the product level: This is a placeholder in the db for future use.
objectbank_pk1 id false true

For random blocks (bbmd_sectiontype=3) this is the pk1 of the canvas (bbmd_asi_type=1) record for the first source pool for the block. If there are multiple pools then they will all be recorded in the qti_block_mapping table
