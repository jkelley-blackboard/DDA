This table contains the metadata associated with a report definition.

Foreign key constraints

  1. report_def_fk2

Indexes

    • plugins_pk1
Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
name nvarchar(255) false false

This is the name of the report definition.
title nvarchar(255) false false

This is the title of the report definition.
version nvarchar(255) '1.0.0.0' false false

This is the version of the report definition.
last_run_date datetime false true

This is the date time when the report definition was executed for the last time.
last_run_duration numeric false true

This is the time duration that the report definition was executed last time in milliseconds.
description nvarchar(1000) false true

This is the description of the report definition.
source nvarchar(255) false true

This is the souce of the report definition. All system shipped report definitions have source of Blackboard Inc.
package_name nvarchar(255) false false

This is the report engine package name that wraps around one or a group of report definitions.
catalog_file nvarchar(255) false true

This is the catalog file that used for JReport. It should be be deprecated since JReport is no longer used.
definition_file nvarchar(255) false false

This is the code file that contains the logic of the report definition which is executed at time of report execution.
imported_date datetime false true

This is the date time when the report definition was imported into the system.
parameter_handler nvarchar(255) false true

This is a java class that handles custom parameters at execution point.
dtmodified datetime false true

This is the date time when the report definition was last modified.
provider varchar(25) 'BIRT' false false

This is the report engine (provider). After the migration to BIRT report engine is complete, this column should have the value of BIRT across all report definitions.
plugins_pk1 id false true

This is a foreign key referencing the primary key of the plugins table.
scheduled_ind char(1) 'N' report_def_scheduled_ind
  • Y
  • N
false false

This is a boolean flag (Y/N) that indicates whether the report definition execution will always be scheduled.
system_avail_ind char(1) 'Y' report_def_sys_avail_ind
  • Y
  • N
false false

This is a boolean flag (Y/N) that indicates whether the report definition is generally available in the system.
prev_system_avail_ind char(1) 'Y' report_def_sys_avail_ind2
  • Y
  • N
false false

The previous configured value of system_avail_ind. This is used when enabling/disabling Building Blocks so that when a Building Block is disabled, then enabled, the previous customizations are retained.

context_avail char(1) 'A' report_def_avail_ind
  • A
  • S
  • C
false false

This indicates the availability of the report definition to contexts. A = All Places, S = Selected Places, C = Selected Places and All their decendants.
