ews_notification

This table contains a record for each retention center notification sent. There will be only one record in this table even if the notification was sent to more than one student.

Primary key constraint

ews_notification_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. ews_notification_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_users table, referential action set null on delete.

    • sender_course_users_pk1
  2. ews_notification_fk3 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].course_main table, referential action cascade on delete.

    • course_main_pk1

Indexes

  1. ews_notification_ie1
    • notification_date
  2. ews_notification_if2
    • sender_course_users_pk1
  3. ews_notification_if3
    • course_main_pk1
Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
notification_date datetime NOW false false

Date and time of the ews notification.
notification_type varchar(10) 'EMAIL' false false

This is the type of notification, e.g.: email.
rule_type varchar(25) false true

This is ews rule type.
notification_subject nvarchar(300) false true

This is the subject line of the notification if it is modified by the instructor from the canned subject.
notification_text ntext false true

This is the text of the notification if it is modified by the instructor from the canned message.
text_format_type char(1) false true

This is the text format for VTBE. P=Plain text, H=HTML, S=Smart Text, X=Default.
sender_course_users_pk1 id false true

This is a foreign key to course_users.pk1. It is the course_users_pk1 of the person who sent the notification.
sender_user_name nvarchar(50) false true

This is the user name of the person who sent the notification. In the case of instructor removed from the course, sender_course_users_pk1 will set null, but this column data will remain for history information.
course_main_pk1 id false false

This is a foreign key referencing the course_main.pk1. It is the course that the notification is sent from.
