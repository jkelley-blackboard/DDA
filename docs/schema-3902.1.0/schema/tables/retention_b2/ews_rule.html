This is a boolean flag indicating if the rule is to be compared to course average. Default value=N.
rule_type_spec varchar(2) false true

This field is used to identify the implementation of the rule type for the rule instance. For example, for due date rule, there are 3 implementations currently. One is on a specific grade book item, one is on any number of gradebook item, one is for course based activity. We would have different values in this field acting as metadata for the type.
