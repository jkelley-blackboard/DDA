eud_item_recipient

This table contains a list of recipients who will or have received a notification about a notification item.

Primary key constraint

eud_item_recipient_pk

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. eud_item_recipient_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the eud_item table, referential action cascade on delete (cascade).

    • eud_item_pk1
  2. eud_item_recipient_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (cascade).

    • user_pk1
  3. eud_item_recipient_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the groups table, referential action cascade on delete (setnull).

    • group_pk1
