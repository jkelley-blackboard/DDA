eud_method_setting

This table represents a notification method setting, which may be for an administrator course or organization, a user global course or organization, or a user-specific course or organization setting. This table defines settings that control whether notifications for specific nautilus event types are enabled for each of the various kinds of distribution mechanisms. Like its general settings counterpart, it allows for hierarchies - which is to say, a method setting can be specified at both the admin the user level, with the latter overriding the former.
