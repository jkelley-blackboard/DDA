clp_section_aff_filter

This table contains context based section affiliation list filter criteria for a deployment. These criteria are used to produce section and ee section affiliation lists when the deployment is actually started. Only one of the clp_program_pk1, clp_course_pk1, clp_ee_pk1, clp_crs_section or clp_ee_section_pk1 columns will be filled out for each record.
