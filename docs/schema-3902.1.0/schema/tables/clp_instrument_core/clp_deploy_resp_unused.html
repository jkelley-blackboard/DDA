clp_deploy_resp_unused

This table contains response records that are moved from clp_deployment_responses table at the end of a deployment for responses that are not received.

Primary key constraint

clp_deploy_resp_unused_pk1

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. clp_deploy_resp_unused_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the deployment table, referential action cascade on delete (cascade).

    • deployment_pk1
  2. clp_deploy_resp_unused_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the clp_section table, referential action cascade on delete (setnull).

    • clp_section_pk1
  3. clp_deploy_resp_unused_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the clp_program table, referential action cascade on delete (setnull).

    • clp_program_pk1
  4. clp_deploy_resp_unused_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the clp_ee_section table, referential action cascade on delete (setnull).

    • clp_ee_section_pk1
  5. clp_deploy_resp_unused_fk5 (On Delete SetNull)

    This is a foreign key referencing the primary key of the clp_operating_unit table, referential action cascade on delete (setnull).

    • clp_ou_pk1
  6. clp_deploy_resp_unused_fk6 (On Delete SetNull)

    This is a foreign key referencing the primary key of the clp_person table, referential action cascade on delete (setnull).

    • clp_person_pk1
  7. clp_deploy_resp_unused_fk7 (On Delete SetNull)

    This is a foreign key referencing the primary key of the materialized_list table, referential action cascade on delete (setnull).

    • materialized_list_pk1
  8. clp_deploy_resp_unused_fk8 (On Delete SetNull)

    This is a foreign key referencing the primary key of the [AS_CORE].course_main table, referential action on delete set null.

    • crsmain_pk1
