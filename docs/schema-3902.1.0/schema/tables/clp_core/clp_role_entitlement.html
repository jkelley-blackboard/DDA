clp_role_entitlement

This table lists all entitlement that each outcomes role holds.

Primary key constraint

clp_role_entitlement_pk1

This is the name of primary key constraint on columns clp_role_pk1 and entitlement_uid.

  • clp_role_pk1
  • entitlement_uid

Foreign key constraints

  2. clp_role_entitlement_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the entitlement table, referential action cascade on delete (cascade).
