prtfl_cntnt

Records in this table indicate the mapping between the Portfolio and content area. Specifically a Portfolio page can contain one or more Content areas.

Primary key constraint

prtfl_cntnt_pk1

  • pk1

Foreign key constraints

  1. prtfl_cntnt_fk1

    Do not cascade delete here - if a cntnt item (artifact) is being used by a portfolio (i.e. row in this table) then we do not want to allow the deletion of that cntnt row

    • cntnt_pk1
  2. prtfl_cntnt_fk2 (On Delete Cascade)
    • prtfl_page_pk1

Indexes

  1. prtfl_cntnt_if1
    • cntnt_pk1
  2. prtfl_cntnt_if2
    • prtfl_page_pk1

A unique (generated) identifier for this portfolio artifact (i.e. an artifact added to a portfolio) used to name the xythos internal folder storing the supporting files. Will be null if artifactProviderId is null on the associated cntnt row.

When a portfolioArtifact is created the user is given artifact-specific options to tune the display of the artifact. This is only set when uuid is not null and the related row in cntnt has a non-null artifactProviderId
