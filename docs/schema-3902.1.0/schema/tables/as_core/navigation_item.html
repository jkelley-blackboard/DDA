Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
application varchar(64) false true

Application accessed through the navigation item. Application module, e.g: instructor_gradebook, course_email, discussion_board, etc.
internal_handle varchar(255) false false

Internal handle.
entitlement_uid varchar(255) false true

Defines the entitlement required to access the item. Entitlement required to use this navitem. There are many entitlements in the entitlement table to chose from, e.g., system.admin.VIEW, Modify, etc
family varchar(255) false true

Top level for groupings of navigation items. Used to group navigation items together
href varchar(500) false true

Address of the application in the file system. URL to access the target item.
enabled_mask int false true

This the bit mask that determines if the tool is enabled or not. NAVITEM_COURSE = 0x00000001, NAVITEM_ORG = 0x00000002, BBSUPPORT_ONLY = 0x000000040, NONE = 0x00000000, DISABLED = 0x00000020 ALL = 0xffffffff.
component_type varchar(20) false true

Groups items under the Sub-Group heading. Possible values: ADMIN_ENTRY, CHILD, COURSE_ENTRY, CP_ENTRY, SYSTEM_ENTRY, MENU_ITEM, NONE
navigation_type varchar(20) false true

Course or System Determines whether the navigation item is available in courses or through the portal. Indicates where the navitem will be used. Possible values: GROUP, COURSE, SYSTEM
src varchar(255) false true

Image or icon connected to the navigation item.
label varchar(255) false true

Name of the navigation item as it appears within the system. User-visible label - may be a key to a language string.
description varchar(3900) false true

User-visible description - may be a key to a language string.
sub_group varchar(255) false true

Groups navigation items under the Family heading.
position int false true

Order that the navigation item appears in the system in relation to other items in the same grouping.
pk1 id true false

This is the surrogate primary key for the table.
launch_ind char(1) 'N' indicator83
  • Y
  • N
false true

Determines whether or not to launch the application in a separate browser window. if 'Y', the anchor tags target attribute will have a value of _blank, otherwise it will be _self
redirect_url varchar(255) false true

Some navigation items, such as the Lost Password page, may be redirected. If allowed, the redirected URL appears in this field. This is navigation redirect url. If set it will override the href (Some license levels let the administrator configure this)
user_label nvarchar(255) false true

This is customized label. If set it will override the label.
visible_ind char(1) 'Y' false true

Determines whether or not the item is visible to users. if 'N', this is equivalent to disabling the nav item, but not in all situations.
