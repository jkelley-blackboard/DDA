content_handler_type

Type mapping for content handlers. Each content handler can have one or more type mappings. The content handler type is used to organize content handlers into groups.

Primary key constraint

content_handler_type_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. content_handler_type_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].content_handlers table, referential action cascade on delete.

    • content_handlers_pk1

Indexes

  1. content_handler_type_ak1

    Ensures that a content handler is only mapped to a given action type at most once.

    • content_handlers_pk1
    • action_type
  2. content_handler_type_if1

    Index for searching content handlers based on action type

    • action_type
