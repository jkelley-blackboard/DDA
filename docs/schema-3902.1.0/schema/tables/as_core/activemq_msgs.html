activemq_msgs

This is the table where ActiveMQ persistent messages are stored. Indexes were named on purpose after the convention of ActiveMQ's so that they can be manipulated, if necessary, by ActimveMQ broker's DDL statement executions. Ref.: http://activemq.apache.org/maven/[activemq_ver]/apidocs/src-html/org/apache/activemq/store/jdbc/Statements.html
