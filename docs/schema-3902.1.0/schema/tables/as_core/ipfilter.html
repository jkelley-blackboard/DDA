ipfilter

Each Web Service or Web Service client can have a set of IP Filters associated with it. This table stores filters for both use cases (so one of ws_service_pk1 or ws_client_pk1 should be non-null)

Primary key constraint

ws_ipfilter_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. ws_ipfilter_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].ws_service table, referential action cascade on delete.

    • ws_service_pk1
  2. ws_ipfilter_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].ws_client table, referential action cascade on delete.

    • ws_client_pk1
  3. ipfilter_profile_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].ipfilter_profile table, referential action cascade on delete.

    • ipfilter_profile_pk1
