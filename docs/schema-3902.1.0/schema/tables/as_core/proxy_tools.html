proxy_tools

Each proxy tool (with placements) will have an associated row in this table. When the proxy tool is deleted, this row is deleted. Most of the information in this table is essentially a 'cache' of data taken from the proxy tool's XML (stored in ws_client.description)

Primary key constraint

proxy_tools_pk

This is the name of primary key constraint on column pk1.

Foreign key constraints

  1. proxy_tools_fk1 (On Delete Cascade)
