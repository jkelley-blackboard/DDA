permission varchar(20) false true

This is the action that the entitlement has permission to perform.
label nvarchar(255) false true

This is the label for the entitlement that will show up in the Privileges list. Used in conjunction with entitlement_type.
entitlement_type varchar(100) false true

This is the entitlement type. Valid values are 'Course', 'System', and 'Personal'. If non-null, used for populating the Privileges list for course/system roles ('Personal' is used for the 'None' system role).
