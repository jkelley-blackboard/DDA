attribute_definition

An AttributeDefinition contains the meta-data describing an attribute of an entity within the system. These attributes can be user-defined or system attributes. The template and form framework uses these definitions to map fields to their attributes, and the persistence system uses these definitions to determine where to load and store an entity's attribute values.
