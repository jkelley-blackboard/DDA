Bit field for session metadata: * First bit - Sesssion folder structure 1 = new session folder with BbSessionIdMd5 structure 0 = new session folder with BbSecureSessionIdMd5 structure * Second bit - Session type 1 = mobile 0 = web * Third bit - Impersonation session type 1 = impersonated 0 = non impersonated When null indicates old structure session and undetermined session type, may be mobile or web. Majority of sessions will have a value of 1 (web session using the new session folder with BbSessionIdMd5 structure).
