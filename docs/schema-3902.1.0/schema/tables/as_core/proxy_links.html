proxy_links

Each http-action aka link declared by a proxy tool will have an associated row in this table.

Primary key constraint

proxy_links_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. proxy_links_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].proxy_tools table, referential action cascade on delete.

    • proxy_tools_pk1
  2. proxy_links_fk2

    This is a foreign key referencing the primary key of the [AS_CORE].content_handlers table.

    • content_handlers_pk1

Indexes
