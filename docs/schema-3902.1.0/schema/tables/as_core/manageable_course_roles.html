manageable_course_roles

This table maps the course roles which can be managed by each role in the course_roles and system_roles table

Foreign key constraints

  1. manageable_course_roles_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].system_roles table.

    • system_role
  2. manageable_course_roles_fk2

    This is a foreign key referencing the primary key of the [AS_CORE].course_roles table.

    • course_role
  3. manageable_course_roles_fk3

    This is a foreign key referencing the primary key of the [AS_CORE].course_roles table.

    • manageable_course_role

Unique constraints

  1. manageable_course_roles_uk
    • course_role
    • system_role
    • manageable_course_role

Indexes

  1. manageable_course_roles_ak
    • pk1
  2. manageable_course_roles_if1
    • system_role
  3. manageable_course_roles_if2
    • course_role
  4. manageable_course_roles_if3
    • manageable_course_role
