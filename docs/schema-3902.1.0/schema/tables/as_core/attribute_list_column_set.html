attribute_list_column_set

This table represents a collection of active list columns. Active columns are the subset of available columns that are displayed in a particular column configuration. There are two kinds of lists. System Default is the default set of columns at the entity type level. All lists that belong to this type and do not exist in an operating unit context will use this set as their default. Customized is the set of columns that has been customized by a user, at either the system or ou-level (operating unit level).
