queued_tasks

This table contains tasks that were scheduled for delayed execution.

concurrency_group varchar(255) false true

Concurrency Group of the task. It can be anything. It could be the same as task_type, which would be the case if the object was created with a QueueTask java object and this column was not explicitly set to differ from task_type. When this column was created, 'C_Background' was one instance that's not a task_type but a concurrency_group, and it was meant for a special 'C' task_type that's run as a background task.
