ws_tool_entitlements

When a tool with placements requests entitlements they are stored in this table. The tool will request them based on webservice name+operation name. During registration these operations are mapped to a list of raw entitlements for this table.

Primary key constraint

ws_tool_entitlements_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. ws_tool_entitlements_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].ws_client table, referential action cascade on delete.

    • ws_client_pk1
  2. ws_tool_entitlements_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the [AS_CORE].entitlement table, referential action cascade on delete.

    • entitlement_uid
  3. ws_tool_entitlements_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the [AS_CORE].plugins table, referential action setnull (but will be deleted via the plugins table's ondelete trigger).

    • plugins_pk1

    • entitlement_uid
  3. ws_tool_entitlements_if3
