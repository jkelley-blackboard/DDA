videv_video

This table contains the relationship between a video and a content_owner. Before the content owner is persisted, session_id must be populated. Once the content_owner_pk1 is set, session_id must be cleared.

Primary key constraint

videv_video_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. videv_video_fk1

    This is a foreign key referencing the primary key of the [VIDEO-INTEGRATION].content_owner table.

    • content_owner_pk1
  2. videv_video_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the [AS_CORE].users table.

    • author_user_pk1

Indexes

    • session_id

    • content_owner_pk1
