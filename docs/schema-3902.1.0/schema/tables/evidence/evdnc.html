evdnc

This table contains general information on Outcomes Artifacts. Outcomes Artifacts originally named Evidence Areas provide a container for text, media and file content. Outcomes Artifacts may exist independently or within an Outcomes Portfolio.

Primary key constraint

evdnc_pk1

This is the name of primary key constraint on column pk1.

  • pk1

  2. evdnc_fk2 (On Delete SetNull)
