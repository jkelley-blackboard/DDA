wiki_stream_event

Stores wiki related stream events to populate the My Blackboard stream viewer

Primary key constraint

wiki_stream_event_pk

This is the name of primary key constraint

  • pk1

Foreign key constraints

  1. wiki_stream_event_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the wiki table, referential action cascade on delete (cascade).

    • wiki_pk1
  2. wiki_stream_event_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • actor_pk1
  3. wiki_stream_event_fk3 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • actee_pk1
  4. wiki_stream_event_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_main table, referential action cascade on delete (setnull).

    • crsmain_pk1
  5. wiki_stream_event_fk5 (On Delete SetNull)

    This is a foreign key referencing the primary key of the groups table, referential action cascade on delete (setnull).

    • group_pk1
  6. wiki_stream_event_fk6 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_contents table, referential action cascade on delete (setnull).

    • course_contents_pk1

Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
actor_pk1 id false true

FK to users.pk1 for the person doing the action - i.e. a comment on a wiki page would set the commenter in actor_pk1. This column will be set to NULL if the user was deleted in which case this row should be ignored.
actee_pk1 id false true

FK to users.pk1 for the person the action was done against - i.e. a comment on a wiki page would set the post-author in actee_pk1. This will be NULL if the stream event is not in response to a particular user or if the user was deleted.
crsmain_pk1 id false true

FK to course_main for the course in which the event occurs. Will be set to NULL if the course was deleted.
group_pk1 id false true

FK to groups for the group in which the event occurs. This will be NULL for non group wikis or if the group was deleted.
event_date datetime false false

Date and time the event occurred.
course_contents_pk1 id false true

FK to course_contents.pk1 if the event pertains to a course contents item. This will be NULL if the event does not have a course_contents.pk1 or if the course content was deleted.

wiki_pk1 id false false

Wiki entry pageGuid - This will be NULL if the event has nothing to do with wiki entry. This is made up of 32 character hex digits created from UuidFactory.
wiki_entry_version int false true

Wiki entry version - This will be NULL if the event has nothing to do with wiki entry.

Wiki title at the time of the event - This corresponds to WIKI.NAME
wiki_entry_title nvarchar(400) false true

Wiki entry title at the time of the event - This will be NULL if the event has nothing to do with wiki comments or if it is a deletion event. This corresponds to XYF_PROPERTIES.SHORT_STRING_VALUE
