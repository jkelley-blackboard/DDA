Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
dtcreated datetime NOW false false

This is the date time when the thread/post was first created.
dtmodified datetime NOW false true

This is the date time when the thread/post was last modified.
posted_date datetime false true

This is the date time when the thread/post was posted.
last_edit_date datetime false true

This is the date time when the thread/post was last edited.
lifecycle varchar(64) 'PUBLISHED' false false

This is the state of the thread/post in the publish life cycle. Valid values are PUBLISHED, DRAFT, MODQUEUE, REJECTED, REJECTED_EXPLANATION, HIDDEN, DELETED and UNAVAILABLE
text_format_type char(1) 'S' indicator_msg_fmt
  • S
  • P
  • H
false true

This is the text format for the thread/post body. S = Smart Text, P = Plain Text, H = HTML.
post_as_annon_ind char(1) 'N' indicator80
  • Y
  • N
false false

This is a boolean flag (Y/N) indicates if the thread/post is annonymous.
cartrg_flag char(1) 'N' indicator79
  • Y
  • N
false false

This is a boolean flag (Y/N) indicates if the thread/post is loaded from a cartrige. A message can be loaded from a cartrige during a content exchange operation.
thread_locked char(1) 'N' indicator82
  • Y
  • N
false false

This is a boolean flag (Y/N) indicating if the thread is locked.
post_first_ind char(1) 'N' msg_post_first
  • Y
  • N
false false

This column indicates if the thread is forced to be post first (Y) in case the forum is set to be thread post first.
deanonymize_ind char(1) 'N' allow_deanonymization
  • Y
  • N
false true

This column indicates whether the thread can be de-anonymized. De-anonymization refers to converting the author of a Thread from being anonymous to that of an Instructor. Only Threads created via Import or Course Copy are eligible for de-anonymization (Y). Threads are not elible for de-anonymization if deanonymize_ind is (N) or (NULL)
hit_count int false true

This is the number of times this thread/post is viewed.
subject nvarchar(300) false true

This is the subject line of the thread/post.
posted_name nvarchar(255) false true

This is the user who posted the thread/post.
linkrefid varchar(255) false true

This is a link to a gradable item (gradebook_main.linkrefid) if this is a graded thread.
msg_text ntext false true

This is the text body of the thread/post.
body_length int false true

This is the length of the user visible portion of the thread/post body.
users_pk1 id false true

This is Foreign Key referencing users table. It contains the PK of the user who posted the thread/post.
forummain_pk1 id false false

This is Foreign Key referencing forum_main table. It contains the PK of the forum the thread/post is posted to.
msgmain_pk1 id false true

This is self referencing Foreign Key referencing msg_main table. It contains the PK of the thread/post that this thread/post is replying to.
thread_pk1 id false true

This is self referencing Foreign Key referencing msg_main table. It contains the PK of the seed thread that this message is belonging to.
top_level_response_pk1 id false true

This is self referencing Foreign Key referencing msg_main table. It contains the PK of the top level response (the message whose parent is a thread) that this message is belonging to.
last_update_users_pk1 id false true

This is Foreign Key referencing users table. It contains the PK of the user who edit the thread/post last.
is_spam char(1) msg_is_spam
  • Y
  • N
false true

This indicates if the message has been flagged as spam. A null value means the message has not been evaluated for spam.
groups_pk1 id false true

This is a Foreign Key referencing the groups table. It contains the PK of the group to which this thread/post belongs within a given forum.
ai_state char(1) msg_ai_state_ck
  • N

    Not generated by AI

  • Y

    Generated by AI, not yet accepted by a human

  • O

    Originally generated by AI but accepted by a human

 false true

The user who accepted the AI-generated message
