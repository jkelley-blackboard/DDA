er_task_item_set_item

This table keeps track of which items within an item set (evidence set) where collected by each task. This is mapped to er_item_set_item (as opposed to er_item) because a user can manually remove an item from the evidence set (when an item is removed from an evidence set it is not removed from the repository itself). Besides, we care how an item made it into an evidence set, not how it made it into the repository.
