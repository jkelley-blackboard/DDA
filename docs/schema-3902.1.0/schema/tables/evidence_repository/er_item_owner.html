er_item_owner

This table stores references to the Outcomes entities that can use an item. Each item can be associated with one or more Outcomes Work Contexts, Course Sections, or EE Sections.

Primary key constraint

er_item_owner_pk1

  • pk1

Foreign key constraints

  1. er_item_owner_fk1 (On Delete Cascade)
    • er_item_pk1
  2. er_item_owner_fk2 (On Delete Cascade)
    • work_ctx_pk1
  3. er_item_owner_fk3 (On Delete Cascade)
    • clp_section_pk1
  4. er_item_owner_fk4 (On Delete Cascade)
    • clp_ee_section_pk1
