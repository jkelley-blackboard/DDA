er_sample_set_item_sample

The actual samples contained within a sample set. This is mapped to er_item_set_item_sample (as opposed to er_sample) because not all samples for a given item are included in an evidence set combined with the need for referential integrity should the user decide to manually remove an item from the evidence set (when an item is removed from an evidence set it is not removed from the repository itself).
