Blackboard Learn™ Database Documentation

Welcome

Welcome to the Database Schema documentation for Blackboard Learn. This documentation ONLY applies to the 3902.1.0 version of Blackboard Learn. There is no guarantee that it will be accurate for any future version of Blackboard Learn.

What is it?

This documentation set describes the data characteristics of the tables and columns in the Blackboard Learn database: including datatypes and sizes, nullability, index, sequence, key and constraint information. Additional commentary is also provided for key tables and columns.

What is it not?

It is not a complete description of every database object in Blackboard Learn. Notably absent:

  • Detailed commentary for some tables and columns is not provided in this version.
  • No table or column information for the Xythos engine file storage schema is provided in this version.
  • No information about stored procedures, views, triggers, or database jobs is provided in this version.

It is not yet available in languages other than English.

It is not a guarantee that these tables and columns will continue to be viable, or that they will retain their current characteristics in future releases of Learn. This documentation ONLY applies to the 3902.1.0 version of Blackboard Learn.

Who is it for?

This documentation is intended for intermediate to expert database administrators, as an aid to troubleshooting, capacity planning and analytics.

How do I navigate?

The Learn tables are organized into schemas that roughly correspond to different features or feature sets in Learn. You can access the detailed documentation page for a given table in one of two ways:

  • By clicking on a specific schema in the list in the top right. This will show you a description of the schema, and a list of that schema's tables in the lower right.
  • By clicking on the All Database Objects link. This will display every table in the list in the lower right.
