etl_mat_view_time_vw

This view contains a record for each view in etl_materialized_view table, with a last successful delta ETL run timestamp. The timestamp is set to 1970-01-01:00:00:00 if reset_ind is Y for the view. This view is materialized by the stats ETL process to a stats table. This view is materialized by the stats ETL process to a stats table.
