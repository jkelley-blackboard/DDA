ods_deployment_dim_vw

This view contains a record for each deployment of any instrument. It contains the instrument_pk1 of the instrument that was deployed. This actually concatenates instrument type (A=Artifact Template, P=Portfolio Template,C=Course Eval, S=Survey,O=Offline Instrument) and pk1 so is unique across instruments. Also contains a curr_instr_version_pk1 column which identifies the current version of the deployed instrument. This can come into play when an instrument is modified after a given deployment. Also contains a place_pk1 column which identifies the work context that this deployment belongs to. This column is used to select deployments that belong to the work context where a report is being run from. This view is materialized by the stats ETL process to a stats table.
