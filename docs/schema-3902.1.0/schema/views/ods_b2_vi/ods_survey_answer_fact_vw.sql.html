ods_survey_answer_fact_vw

This view contains a record for each answer (i.e. response) to any question contained in an outcomes survey or course evaluation. This is a fact view without an actual fact except in the case where the question is open-ended. In that case this record will contain the actual answer in it's answer_text column. Otherwise the answer is indicated by the foreign key to ods_answer_dim_vw. This view is materialized by the stats ETL process to a stats table.
