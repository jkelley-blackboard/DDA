lrn_stds_doc

This table contains Standards Documents, the group of standards identified by the combination of discipline, standards document set (and its provider) and publication year. The Blackboard XML format for standards describes a format which contains one standards document per XML file.

batch_uid nvarchar(256) false false

This is a vendor-generated ID which is unique across all standards document records. If a provider publishes a new standards document (same standards document set and discipline but more recent publication year) the vendor will generate a new batch_uid for the newly published standards document.
