lrn_standard

This table contains the attributes for a learning standard which are not shared with other SOGs. Each learning standard will have one clp_sog record and one lrn_standard record.

Primary key constraint

lrn_standard_pk1

This is the name of the primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. lrn_standard_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the clp_sog table, referential action cascade on delete (cascade).

    • clp_sog_pk1

Indexes

  1. lrn_standard_if1
    • clp_sog_pk1
  2. lrn_standard_ie1
    • label
  3. lrn_standard_ie2
    • import_date
