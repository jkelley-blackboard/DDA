prtfl_style

Records in this table indicate the style elements applied to the portfolio.

Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
color_theme varchar(50) false true

This is the name of the color_theme used for this style. Corresponds to a folder name in the portfolio b2 under webapp/previewcss/colors. If null or referencing an non-existent folder then a default color theme will be used
