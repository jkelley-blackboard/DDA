cntnt

This table contains information on content area. Personal Artifacts originally named Content Areas provide a container for text, media and file content. They can be embedded in Personal or Outcomes Portfolios or exist independent of a container. They are basically pieces of content (text + files) that are created and maintained independently inside the content system tab, but they are useful only as something that can be included in portfolios.
