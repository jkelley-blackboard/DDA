prtfl_page

Records in this table are for a single Portfolio page (sp14) or for a SECTION within a prtfl_page_coll(post-sp14). The page maintains a reference to the template from which it was created if it was created from a template.

Primary key constraint

prtfl_page_pk1

  • pk1

Foreign key constraints

  1. prtfl_page_fk1

    No longer on-delete cascade because each page will be related to a prtfl_page_coll row which will be deleted and then this row will be cascade-deleted based on that.

    • prtfl_pk1
  2. prtfl_page_fk2 (On Delete SetNull)
    • prtfl_page_tmplt_pk1
  3. prtfl_page_fk3 (On Delete Cascade)
    • prtfl_page_coll_pk1

Indexes

  1. prtfl_page_if1
    • prtfl_pk1
  2. prtfl_page_if2
    • prtfl_page_tmplt_pk1
  3. prtfl_page_if3
    • prtfl_page_coll_pk1
