clp_content_alignment

This table contains alignments between pieces of content (represented by clp_alignable_content records) and SOGs.

Primary key constraint

clp_content_alignment_pk1

This is the name of the primary key constraint on the column pk1.

  • pk1

Foreign key constraints

  1. clp_content_alignment_fk1 (On Delete SetNull)

  2. clp_content_alignment_fk2 (On Delete Cascade)

    This is a foreign key referencing the primary key of the clp_alignable_content table, referential action cascade on delete (cascade).
