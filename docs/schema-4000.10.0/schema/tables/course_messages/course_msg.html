course_msg

This table contains a snapshot of the information when Course and Ultra messages are sent, and can support messages sent outside the course context.

Primary key constraint

course_msg_pk

This is the primary key of this table.

  • pk1

Foreign key constraints

  1. course_msg_fk1 (On Delete Cascade)

    This is a foreign key referencing pk1 of the conversation table.

    • conversation_pk1
  2. course_msg_fk2 (On Delete SetNull)

    This is a foreign key referencing pk1 of the users table.

    • sender_users_pk1
  3. course_msg_fk3

    This is a foreign key referencing pk1 of the course_main table.

    • crsmain_pk1

Indexes

  1. course_msg_if1
    • conversation_pk1
  2. course_msg_if2
    • sender_users_pk1
  3. course_msg_if3
    • crsmain_pk1
Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the primary key of this table.
body ntext false true

This is the text body of the message. If type == 'S', the body contains a JSON Object with the new participants that were newly added to an Ultra Conversation.
subject nvarchar(200) false true

This is the subject line of the message. While this field is 200 characters, the UI will still restrict the size to 50 characters.
sender_users_pk1 id false true

This is a foreign key to the users table, i.e. users.pk1, and represents the sender of the message.
receivers ntext false false

A JSON Object containing the TO, CC, BCC list of participants, each represented using user.pk1
conversation_pk1 id false true

This is a foreign key to the conversation table, i.e. conversation.pk1. This applies only to ULTRA and is used to group messages for a particular conversation.
sent_date datetime NOW false false

This is the date time when this message was sent
type char(1) 'N' course_msg_type_ck
  • N

    'Normal' message, which represents a message sent by a course user. All Course Messages are 'Normal', while Ultra Messages can be either.

  • S

    'System' or auto-generated messages, which only apply to Ultra, e.g. 'Joe added Tom to the conversation'.

  • A

    'AI' or ai-generated messages, which only apply to Ultra, e.g. 'ADA responded to the convo'.

 false false
crsmain_pk1 id false true

This is a foreign key to the course_main table, i.e. course_main.pk1. This is NULL for messages sent outside a course (future requirement).
content_ids jsonb false true

This is a JSON object containing content IDs associated with the ai answer.
