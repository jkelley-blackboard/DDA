plugin_activity_history

This table stores the activity information related to B2 installation/uninstallation only, we do not store activity related to status changes, and not if the action is part of a Learn install or upgrade.

Foreign key constraints

  2. plugin_activity_history_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table.

    • users_pk1
