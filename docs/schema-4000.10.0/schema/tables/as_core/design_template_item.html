design_template_item

This table contains the default menu items for courses (or organizations - usage depends on template_uid). Used to create course_toc entries.

Primary key constraint

design_template_item_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

    • internal_handle
  2. design_template_item_fk1

    This is a foreign key referencing the primary key of the [AS_CORE].design_template table.

    • template_uid

Indexes

    • template_uid
  2. design_template_item_if2
    • internal_handle
