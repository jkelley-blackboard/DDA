groups_stage

This table stores temporary (staging) data for groups during creation or update operations.

Primary key constraint

groups_stage_pk

This is the name of primary key constraint on column pk1 and session_id.

  • pk1
  • session_id

Foreign key constraints

  1. group_stage_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the sessions table.

    • session_id
    • [sessions].session_id (foreign key reference)
  2. group_stage_fk2

    This is a foreign key referencing the primary key of the sessions table.

    • crsmain_pk1

Indexes

  1. groups_stage_ie1
    • session_id
  2. groups_stage_if1
    • crsmain_pk1
