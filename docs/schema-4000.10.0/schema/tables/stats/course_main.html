asmt_upgrade_version numeric 0 false false

The assessment data has to go through a number of runtime upgrade steps before assessments are usable in a course. This field records the current upgrade-state of the assessments data within this course.
asmt_upgrade_flags numeric 0 false false

After any import or restore operation the asmt_upgrade_version is set back to 0 and any new data re-upgraded. There are certain parts of the upgrade which do not have to run in this case (i.e. the upgrade is only required once per course, not after each import operation). This field is used to store those flags to avoid re-running pieces of the upgrade
