etl_materialized_view

This table lists all views in stats database that will be materialized to physical tables during ETL framework data refresh process.

Primary key constraint

etl_materialized_view_pk

  • pk1

Indexes

  1. etl_materialized_view_ie1
    • view_name
Column name Data type Default value Value constraint Default constraint Identity? Nullable? Description
pk1 id true false

This is the surrogate primary key for the table.
view_name varchar(30) false false

This is the name of the view that will be materlized to physical table.
etl_sequence int false false

This is the squence in which the views will be materlized. It is user set.
datecreated datetime NOW false true

This is date time that the view was added.
delta_sproc_name varchar(30) false true

The name of the stored procedure that should be used to process the _tp table for this view that contains delta rows. Procedure needs no args.
reset_ind char(1) 'Y' false false

This is a boolean flag (Y/N) indicates if this view (which is normally processed using deltas) should be replaced completely.
batch_size int 1000000000 false true

This is the number of records to process each batch. It can be set differently by view.
unique_sort varchar(500) false true

The names of the columns that are used as unique key for merge delta.
