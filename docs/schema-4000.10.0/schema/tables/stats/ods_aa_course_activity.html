ods_aa_course_activity

This table is a result of the materialization of ods_aa_course_activity_vw. The data is loaded through ETL process. It lists course access records aggregated at the course level within each login session.

Primary key constraint

ods_aa_course_activity_pk

  • login_pk1
  • course_pk1

Indexes

  1. ods_aa_course_activity_ie1
    • user_pk1
  2. ods_aa_course_activity_ie2
    • course_pk1
  3. ods_aa_course_activity_ie3
    • course_pk1
    • user_pk1
