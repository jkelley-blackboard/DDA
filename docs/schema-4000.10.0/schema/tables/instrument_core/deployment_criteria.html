This value defines the type of the criterion represented in this row and essentially identifies which of the foreign key references -- mi_node_pk1, crsmain_pk1, term_pk1, course_role, institution_roles_pk1, and users_pk1 will be set. In each row, only one of those references can be set. The type is set when the record is created. If the referenced record is deleted, the reference will be set to null but the type will remain to identify the type of the original reference.

This value will remain after the reference itself has been set to null and provides a way to identify the entity that was deleted.
