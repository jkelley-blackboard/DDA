clp_ee_section

This table records general information on sections that belongs to education experience.

Primary key constraint

clp_ee_section_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. clp_ee_section_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the clp_ee table, referential action cascade on delete (cascade).

    • clp_ee_pk1
  2. clp_ee_section_fk2

    This is a foreign key referencing the primary key of the term table.

    • term_pk1
  3. clp_ee_section_fk3

    This is a foreign key referencing the primary key of the form table.

    • form_pk1
  4. clp_ee_section_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_main table, referential action cascade on delete (setnull).

    • course_pk1
  5. clp_ee_section_fk5 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • modifying_user_pk1
