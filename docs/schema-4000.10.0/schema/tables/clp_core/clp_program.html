clp_program

This table contains general information on programs in outcomes system.

Primary key constraint

clp_program_pk

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. clp_program_fk1 (On Delete Cascade)

  2. clp_program_fk2

    This is a foreign key referencing the primary key of the form table.

    • form_pk1
  3. clp_program_fk3

  4. clp_program_fk4

    This is a foreign key referencing the primary key of the work_ctx table.

    • work_ctx_pk1
  6. clp_program_fk6 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).
