gradebook_grade

Main Grade record to which all attempts are connected. A grade can either be an override (value in this record) or an attempt grade - which attempt is chosen for the grade depends on the Attempt Aggregation Scheme (see gradebook_main.aggregation_model). It describes a cell in the gradebook.
