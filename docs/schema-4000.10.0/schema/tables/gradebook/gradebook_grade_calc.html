gradebook_grade_calc

Calculated Grades at a given point in time (see version) - Not realtime, refresh needs to be triggered from the application tier. Presently this is called prior to Adaptive Release or Early Warning System evaluation in case calculated columns are involved in at least one of those rules for a given course.
