gradebook_log

This table contains log information on grades.

Primary key constraint

gradebook_log_pk

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. gradebook_log_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the gradebook_main table, referential action cascade on delete (cascade).

    • gradebook_main_pk1
  2. gradebook_log_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the course_roles table, referential action on delete set null.

    • modifier_role
