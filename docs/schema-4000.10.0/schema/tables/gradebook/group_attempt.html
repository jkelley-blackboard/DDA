group_attempt

This table holds attempt information for group based attempts such as a group assignment or group based graded interactive tool activity such as graded group blog or wiki.

Primary key constraint

group_attempt_pk

This is the primary key on column pk1 of this table.

  • pk1

Foreign key constraints

  1. group_attempt_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the course_content_group table, referential action cascade on delete (cascade).

    • course_content_group_pk1
  2. group_attempt_fk2 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table, referential action cascade on delete (setnull).

    • users_pk1
  3. group_attempt_fk3 (On Delete Cascade)

    This is a foreign key referencing the primary key of the gradebook_main table, referential action cascade on delete (cascade).

    • gradebook_main_pk1
  4. group_attempt_fk4 (On Delete SetNull)

    This is a foreign key referencing the primary key of the groups table, referential action cascade on delete (setnull).

    • groups_pk1
  5. group_attempt_fk5 (On Delete SetNull)

    This is a foreign key referencing the primary key of the [AS_CORE].qti_result_data table, referential action none on delete.

    • qti_result_data_pk1
  6. group_attempt_fk6 (On Delete SetNull)

    Foreign key referencing the primary key of table group_attempt_activity, to record the current controller

    • locked_grp_atmpt_actvty_pk1
  7. group_attempt_fk7 (On Delete SetNull)

    This is a foreign key referencing the primary key of the users table.

    • ai_user_instr_comments
