evdnc_tmplt

Records in this table indicate the template definition for an Outcomes Artifact. The template stores indicators for the types of artifact input elements that are available to the Outcomes Artifact generated from template.

Primary key constraint

evdnc_tmplt_pk1

This is the name of primary key constraint on column pk1.

  • pk1

Foreign key constraints

  1. evdnc_tmplt_fk1 (On Delete Cascade)

    This is a foreign key referencing the primary key of the work_ctx table, referential action cascade on delete (cascade).

    • work_ctx_pk1
  2. evdnc_tmplt_fk2
    • parent_pk1

Indexes

  1. evdnc_tmplt_if1

    This is a foreign key referencing the primary key of the table.

    • work_ctx_pk1
  2. evdnc_tmplt_if2
    • parent_pk1
